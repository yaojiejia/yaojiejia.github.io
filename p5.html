Splasher

This code creates a particle system in p5.js where particles emit continuously as the mouse moves. Each particle is represented by the Particle class, which has properties like position, velocity, lifespan, size, and color. When the mouse moves, the mouseMoved() function is called, which adds a new particle at the mouse's current position. The draw() function updates and displays all particles, removing them from the system when their lifespan reaches zero. Each particle moves with a random velocity and accelerates downward slightly to simulate gravity. The result is a dynamic particle system that responds to the movement of the mouse, continuously emitting particles as the mouse moves across the canvas.
