Arrow

1. `setup()` function: - This function is a part of p5.js and is called once when the program starts. - It is used for initializing settings, creating canvas, creating objects, etc. - In this code, it sets up the canvas and initializes instances of the `Canon` and `Target` classes. 2. `draw()` function: - Also a part of p5.js, this function continuously executes the lines of code contained within it. - It's typically used for continuous drawing and updating of the canvas. - In this code, it clears the background, displays the cannon, target, score, and updates the positions of the projectiles. 3. `mousePressed()` function: - This function is another p5.js function, which is called every time the mouse is pressed. - It's used here to create a new projectile whenever the mouse is clicked. - It calculates the direction from the cannon to the mouse click and creates a new `Projectile` instance with that direction. 4. `displayScore()` function: - Custom function created to display the score on the canvas. - It uses the p5.js functions `textSize()` and `text()` to draw the score on the canvas at a specific location. - It's called within the `draw()` function to update and display the score continuously. 5. `Canon` class: - This is a constructor function for creating instances of cannons. - It takes initial x and y positions as arguments. - The `display()` method draws the cannon on the canvas using the `rect()` function. 6. `Target` class: - Similar to the `Canon` class, this constructor function creates instances of targets. - The `display()` method draws the target on the canvas using the `rect()` function. - The `randomizeX()` method randomly changes the target's x-coordinate within a specified range. 7. `Projectile` class: - This constructor function creates instances of projectiles. - It takes initial x and y positions, and a direction vector as arguments. - The `update()` method updates the position of the projectile based on its velocity and handles collision detection with the canvas boundaries. - The `display()` method draws the projectile on the canvas using the `ellipse()` function. - The `hits()` method checks if the projectile hits the target. If it does, it returns true, otherwise false.
